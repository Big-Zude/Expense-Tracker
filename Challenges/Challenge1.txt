Create a simple expense tracker that runs in console only
 -create all the objects needed
 -create all the functions required
 -create a summary for the transaction
 -reset the account
 -use template strings for the output(optional)
 
 
e.g
 the objects can be
 name:
 expense:
 income
 
 the functions can be:
 -Account()
 -addExpense()
 -addIncome()
 -AccountSummary
 -resetAccount()

if you have difficults, please check the sample answers so that you can have a better idea of how to go about it.

sample output:
Account for zude has $1769.5 balance. $2000 in income and $230.5 expenses
Account for zude has $0 balance. $0 in income and $0 expenses
